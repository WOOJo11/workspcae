요소의 여백(영역) 관련 속성

        HTML 요소는 총 4가지의 영역으로 구성 되어있다.
        1. content 영역 
            - 요소의 내용이 작성되는 영역
            (시작태그 ~ 종료태그 사이에 작성되는 내용을 생각하면 된다.)

        2. padding 영역
            - content 영역과 border 영역 사이 

        3. border 영역
            -요소의 테두리가 지정되는 영역
            - content보다 바깥쪽에서 content 영역을 감싸고 있음.
        
        4. margin 영역
            - 다른 요소와의 간격을 나타내는 영역

padding , border, margin의 방향성

        padding , border, margin은 방향에 따라 크기를 별도 지정할 수 있다
        
        (방향은 상, 하 , 좌 , 우 4개만 존재)

        - 영역 이름 : 값 1개;
            -> 상,하,좌,우 모두 동일한 크기 지정

            Ex) padding : 30px; margin : 50px; border 10px;

        - 영역 이름 : 값 1 , 값2;  (값 2개 설정)
                    -> 값 1 : 상 하
                    -> 값 2 : 좌 우

        - 영역 이름 : 값1, 값2, 값3;
            -> 값 1 : 상
            -> 값 2 : 좌우
            -> 값 3 : 하

        - 영역 이름 : 상 ,우 하 좌 (상 부터 시계 방향)

        - top ,bot, left ,right 상 하 좌 우
        영역 이름 - top : 값;
        영역 이름 - bottom : 값;
        영역 이름 - left : 값;
        영역 이름 - right : 값;

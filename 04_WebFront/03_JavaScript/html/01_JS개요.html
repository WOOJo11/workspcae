스크립트(script) 언어 (== 인터프리터 언어)

    기본 프로그램의 동작을 사용자의 요구의 맞게
    수행되도록 해주는 용도의 언어

    별도로 소스 코드를 컴파일 하지 않고
    인터프리터(interpreter)를 이용하여 소스코드를 한 줄씩 읽어 바로 실행

    -> 컴파일 소요 시간이 없어서 프로그램 실행되는 시간은 빠름
        대신 프로그램 수행 중 소요시간이 늘어남

    짧은 소스코드로 상호작용 하도록 고안됨

Javascript 란?

    웹 브라우저에서 주로 사용하는 
    인터프리터 방식의 객체지향 프로그래밍 언어이다
    (Java와 관련없음)

    ECMA Script
    -(정보와 통신 시스템을 위한 국제적 표준화 기구에서 지정한 표준 스크립트)

JavaScript 작성 방법

  1. 브라우저 Console에 직접 작성
  2. html 태그 내부에 script 태그를 이용해 작성(internal 방식)
  3. htnl 파일 외부에 .js 확장자 파일을 이용해 작성(external 방식)
  4. html 태그에 직접 작성하는 방법 (inline 방식)
